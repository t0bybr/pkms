# Ziel
- Physisches Notizbuch behalten
- Scans landen automatisch in Paperless-ngx
- Aufgaben/ToDos werden per KI extrahiert

# Idee
- Scanner/DIY-Buchscanner → Watch-Folder
- OCR → durchsuchbare PDFs
- Lokale KI (Ollama) extrahiert:
  - ToDos
  - Termine
  - Tags
