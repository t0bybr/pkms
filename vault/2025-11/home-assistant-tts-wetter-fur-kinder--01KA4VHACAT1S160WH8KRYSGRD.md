# Idee
- Jeden Morgen personalisierte Wetteransage
- Einfach erklärt für Kinder (z. B. „Heute brauchst du eine Mütze“)

# Bausteine
- Wetterdaten aus HA-Sensoren
- Template für Text („Wenn Regen > X → Regenjacke-Hinweis“)
- TTS über Google / lokale Engine
- Ausgabe über Nest-Lautsprecher im Kinderzimmer
