# Hardware-Optionen
- Smartphone mit Root + Automatisierung
- Alternativ: ältere Kompaktkamera (Sony RX100 / Canon G-Serie)
- Ständer + LED-Licht, möglichst einfach

# Automatisierung
- Knopfdruck:
  - Licht an
  - Foto
  - Upload in überwachten Ordner (Pi / NAS)
- Paperless-ngx verarbeitet automatisch
