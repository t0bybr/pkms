# Name
- `pkms` als Haupt-CLI

# Subcommands
- `pkms new` – neue Notiz mit Template + UUID
- `pkms ingest` – Dateien einlesen, indexieren
- `pkms search` – BM25 + Embeddings + RRF
- `pkms doctor` – Konsistenzcheck (Links, Frontmatter, etc.)

# Wunsch
- Möglichst wenig Abhängigkeiten
- Konfiguration über einfache YAML/JSON-Datei
